# 📌 ReadMe - Running Notebooks on Vertex AI (Colab Enterprise)

Welcome to the **Python and AI Ops with Vertex AI** course! 🎉  
To run the notebooks without compatibility issues, **import them directly into Vertex AI** and use **Colab Enterprise**.

## 🚀 Instructions for Running Notebooks on Vertex AI

You will have 3 notebooks to complete during this practical session:
- 02a Vertex AI Experiments
- 02b Model Registry
- 02c Hyperparameter Tuning

### 1️⃣ Access Google Cloud Console  
🔗 [Google Cloud Console](https://console.cloud.google.com/)

### 2️⃣ Open Vertex AI Workbench  
- In the menu, go to **Vertex AI > Workbench**
- Select **Colab Enterprise**

### 3️⃣ Import Your Notebook  
- Make sure you are in the `europe-west3` region
- Click on **"Open File"**  
- Select **"Upload File"** and import your `.ipynb` notebook

### 4️⃣ Run Your Notebook  
- Execute the cells one by one and follow the course instructions 📖

Happy exploring and learning with Vertex AI! 🚀💡
